## Como Publicar no GitHub Pages

1. Crie um repositório no GitHub (ex: `selped2026`)
2. Faça upload dos seguintes arquivos na raiz do repositório:
   - `index.html` — renomeie o arquivo `selped2026_stories.html` para `index.html`
   - `README.md` — este arquivo
3. Vá em **Settings → Pages**
4. Em *Source*, selecione **Deploy from a branch**
5. Escolha a branch `main` e a pasta `/ (root)`
6. Clique em **Save**

Em poucos minutos, seu site estará disponível em:

```
https://<seu-usuario>.github.io/selped2026/
```
